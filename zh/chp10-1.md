# PWM Timer 定时器 #
## 定时器工作原理

功能：计时，中断，PWM Timer (驱动PWM信号的设备)

原理：类似于以前的 "沙漏"

	沙漏的计时原理：沙子量，漏沙的速度，到时的反转(连续)

Timer:	

	沙子量： Counter的初值
	漏沙速度：counter-- (自动完成，并且依据Clock=PCLK=66Mhz)
	反转：reload 操作 (InitValue -> Counter)
	装沙子：Manual update
	
真正的硬件设计是怎样的？

	TCNTBn - 用来装沙子的量筒，可以修改
	TCNTn - 真正用来做 counter--, 不可修改，不可见
	TCNTOn - 可以用来观察 TCNTn 的值的变化
